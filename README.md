## 🔨 Things I've Built

### 🦅 [Atlas Guard](https://github.com/Ziad-Yousfi/atlas-guard) — Wildlife Monitoring Platform
> `React` · `Vite` · `REST API`

A real-time dashboard for wildlife monitoring and environmental tracking. Built with a clean React/Vite architecture, featuring live data feeds and interactive map overlays. Conservation meets code.

---

### 🚗 [UniRide](https://github.com/Ziad-Yousfi/uniride) — Student Carpooling App
> `Laravel` · `PHP` · `MySQL`

A carpooling platform designed for university students — matching drivers and passengers, managing trips, and reducing the daily commute headache. Full MVC backend with a smooth, session-aware frontend.

---

### 📊 [SmartTender](https://github.com/Ziad-Yousfi/smarttender) — Tender Intelligence Tool
> `Python` · `Streamlit` · `NLP`

SmartTender automates the analysis and parsing of public tenders. It uses NLP to extract key fields and helps decision-makers move faster through procurement noise.
